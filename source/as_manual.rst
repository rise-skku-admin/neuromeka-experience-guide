A/S Manual
==========

CS Request
-----------------

- Official Neuromeka homepage / Support / CS request

`neuromeka cs <https://www.neuromeka.com/cs>`_

- Fill out the form and submit request
    - Delivery date, Model name, Indy S/N are written in the base of the robot.
    - Check the Status table for ID number.
    - Write down error message in ‘Request information’ part.
- When the counselor tells you about the box, tell them you have the box.
    - Change the angle of the robot to [0,0,160,0,160,0] and place the robot in the box.

.. figure:: as_manual/controller.png
   :alt: controller
   :width: 400px
   :align: center
   
- When the counselor tells you about the Controller, Take a picture of the controller IO part and unplug all cables.
    - The controller does not need to be packed in a box.
